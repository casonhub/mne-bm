Biomarker-Cognition-MND Mapping Tool

This tool has been developed to enable reviewers to validate the findings presented in the paper by interactively exploring the relationships between biomarkers, mental and neurological disorders (MNDs), and cognitive skill impairments

User Guide

Use the links below to explore relationship amoung the biomarker, MNDs and impaired cognitive skills. Click on each section to see the mapping:

  • In Module 1 Select biomarkers from the dropdown menu, then add each biomarker by clicking the + sign. After that, click 'Submit' to view the associated disorders and affected brain regions. You can add more biomarkers one by one to see the list of MNDs common to the set of entered biomarkers.
  • In Module 2 Similar to Module 1, enter a biomarker to view the list of associated cognitive skills affected by changes in the selected biomarker. You can add multiple biomarkers by selecting them one by one.
  • In Module 3 of this application, you can select any MNDs from the dropdown menu to view the list of impaired cognitive skills along with their associated common biomarkers
  • In Module 4 You can select a biomarker or a set of biomarkers by adding them one at a time. Once you submit, you will see a list of mental and neurological disorders along with the associated impaired cognitive skills.
  • In Module 5 You can select one or more impaired cognitive skills and submit them. Upon submission, you will receive a list of associated disorders along with their corresponding biomarkers.
